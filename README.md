\# 📊 Retail Pricing Intelligence Dashboard



\## End-to-End Data Analytics Project



\---



\## 📌 Project Overview



The Retail Pricing Intelligence Dashboard is an end-to-end Data Analytics project developed to analyze pricing strategies, inventory levels, customer ratings, discounts, and product returns for a fashion retail business.



The project transforms raw retail data into meaningful business insights using Python for data cleaning and feature engineering, SQL for business analysis, and Power BI for interactive dashboards.



This project demonstrates the complete workflow of a Data Analyst—from raw data preparation to dashboard creation and business decision support.



\---



\## 🎯 Business Problem



Retail businesses often struggle to answer important questions such as:



\- Which brands have the highest average selling price?

\- Which product categories receive the highest discounts?

\- Which categories generate the highest return rates?

\- Which seasons require higher inventory?

\- Which brands have better customer satisfaction?



Without proper analysis, pricing decisions and inventory planning become difficult.

\## ⚙️ Feature Engineering



To improve the quality of analysis, several new features were created during preprocessing:



\- \*\*Discount Amount\*\* = Original Price − Current Price

\- \*\*Price Band\*\* (Low, Medium, High)

\- \*\*Purchase Month\*\* extracted from Purchase Date

\- \*\*Return Status\*\* (Returned / Not Returned)



These engineered features enabled deeper pricing, inventory, and return analysis.

\## 📈 Key Business Insights



From the analysis, the following insights were identified:



\- Premium brands maintain higher average selling prices than budget brands.

\- Some product categories receive significantly higher markdowns, indicating promotional strategies.

\- Inventory levels vary across seasons, highlighting the need for seasonal stock planning.

\- Certain categories have higher return rates, suggesting opportunities to improve product quality or customer expectations.

\- Customer ratings differ across brands, helping identify high-performing and underperforming brands.



These insights can support pricing decisions, inventory optimization, and customer satisfaction improvements.
